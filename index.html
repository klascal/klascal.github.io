Instellingen

Hoe vind je de koppelcode in Zermelo?
  1. Open het Menu
  2. Klik op Instellingen
  3. Klik op Koppel externe applicatie
  4. De bovenste regel is je Schoolnaam (het deel voor .zportal.nl) en de regel daaronder je Koppelcode
  5. Typ het over in Klascal of scan de QR-code

Hoe koppel je Somtoday?
    Methodes:
    • Sleep de link hieronder naar je bladwijzerbalk:
      Somtoday koppelen
    • Open Somtoday
    • Klik op de bladwijzer
    • Kopieer deze code:
                            
const refreshToken = JSON.parse(
  localStorage[
    "CapacitorStorage." +
      JSON.parse(localStorage["CapacitorStorage.SL_AUTH_CONFIG_RECORDS"])
        .currentSessionIdentifier.UUID
  ]
).refresh_token;
const targetOrigin = "https://klascal.github.io#" + refreshToken;
const newWindow = window.open(targetOrigin);
    • Ga naar Somtoday
    • Open de console:
      Ctrl Shift J (Cmd Option J op Mac)
      of Rechtermuisknop → Inspecteren → Console
    • Plak de code in de console
    • Mogelijk moet je 'plakken toestaan' typen om de code te plakken
    • Klik op Enter
