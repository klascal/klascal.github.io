# <img alt="" height="30" src="https://github.com/klascal/klascal.github.io/blob/e749bd5201fe01c99c2634115d8622e21dcfe77a/logo.svg"> Klascal

**Een _mooie_ rooster webapp voor Zermelo**

### Functies
- Werkt offline
- Laat pauzes/tussenuren zien
- Donkere/lichte modus
- Material 3 Expressive
- Kleurenthema's
- Volledige vaknamen
- Voeg aangepaste CSS toe
- Laat klassennamen zien
- Rooster draaien (Zoals bij Portal)
